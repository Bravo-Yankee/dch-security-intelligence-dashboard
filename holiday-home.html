n8n Automation — HHO WhatsApp Check-In Parser
PRODUCTION PIPELINE

Community supervisors send HHO check-in/out messages to WhatsApp groups. The n8n workflow intercepts each message via Evolution API webhook, uses Groq Llama 3.1 AI to extract structured guest data, logs every record to Google Sheets, and optionally fires WhatsApp alerts for critical situations. This dashboard reads from the same Google Sheet every 5 minutes.

How the HHO Pipeline Works
📱
1. SUPERVISOR SENDS
Community supervisor sends HHO check-in/out message to WhatsApp group. Includes guest name, nationality, passport/EID, apartment, check dates, vehicle details, HHO operator name & contact.
⚡
2. WEBHOOK CAPTURES
Evolution API forwards the message to n8n via webhook. Message Parser extracts sender, timestamp, shift context (AM/PM), and the WhatsApp group source for routing.
🤖
3. AI EXTRACTS DATA
Groq Llama 3.1 reads the raw WhatsApp text and extracts: event type (check-in/out), community, apt number, guest names, nationalities, IDs, vehicle plate, HHO company, operator contact.
📊
4. LOGGED & DISPLAYED
Structured record appended to the HHO tab in Google Sheets. This dashboard auto-fetches every 5 minutes and renders community cards, guest logs, operator directory, and handover reports.
